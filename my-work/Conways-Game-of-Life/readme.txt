1. To launch game open index.html and hit Start
2. You can change game's config settings -> main.js
3. To show grid change borderSize to 1
4. You can configure grid and cell dimensions in main.js
6. To add or remove cells - enter Draw Mode
7. left click to add cells (click + move to add multiple cells at once)
8. right click to remove cells (click + move to remove multiple cells at once)
